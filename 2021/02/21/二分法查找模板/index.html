二分法查找模板

二分法查找模板

主要有下面三种

  • 查找正好等于x的位置
  • 查找第一个大于等于x的位置
  • 查找第一个大于x的位置

查找第一个等于x的位置

查找第一个大于等于x的位置

例如

nums = [1,3,3,3,6], x = 3;

注意下面几个问题

  1. 判断终止的条件是low < high
  2. 该查找的区间应该是一个左闭右开的区间，即[low, high)

查找第一个大于x的位置

例如

nums = [1,3,3,3,6], x = 3;

注意下面几个问题

  1. 判断终止的条件是low < high
  2. 该查找的区间应该是一个左闭右开的区间，即[low, high)
